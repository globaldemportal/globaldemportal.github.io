About

Select 1°×1° tiles and download open global elevation data as GeoTIFF. Six free DEMs are available — pick one under Dataset Options. Coverage differs by mission, so the red boundary lines on the map move when you switch products.

Sources

SRTM / NASADEM — NASA/USGS radar (2000).
ALOS AW3D30 — JAXA optical stereo (2006–2011).
Copernicus GLO-30/90 — ESA's open DEM edited from Airbus TanDEM-X radar (2011–2015); the freely redistributable TanDEM-X-based global DEM.

Tile Naming Convention

Tiles are named by their lower-left corner. N37W122 = lat 37–38°N, lon 122–121°W. Each tile covers exactly 1°×1°. Filenames on disk follow each dataset's own convention.

Coverage Limits

The grid is drawn only over cells that actually have a DEM tile, so open ocean stays blank and can't be selected. Red dashed lines mark the selected product's latitude limit; global products (Copernicus) show none.

Data Access

Tiles download straight from the OpenTopography public mirror as GeoTIFF — no account, no login.
NASA's own Earthdata distribution serves the same data but requires a login and blocks browser requests, so it cannot be used directly from a web page.
